# Karunada Kala

Karunada Kala is an Android application developed using Android Studio and Kotlin.  
The project is designed to promote Karnataka’s art, culture, traditions, and heritage through a simple and user-friendly mobile application.

## Features
- User-friendly interface
- Cultural information display
- Image and content management
- Navigation between multiple screens
- Responsive Android design

## Technologies Used
- Kotlin and java
- Android Studio
- XML
- Firebase
- GitHub

## Modules
- Home Screen
- Login/Register
- Cultural Information
- Image Gallery
- User Navigation

## Project Objectives
- To create awareness about Karnataka culture
- To provide digital access to cultural information
- To improve Android development skills
- To implement modern mobile application features

## Practical Implementation
- Designed UI using XML layouts
- Developed application logic using Kotlin
- Integrated Firebase for backend support
- Tested application on Android devices
- Managed source code using GitHub

## Conclusion
The Karunada Kala project helped in gaining practical knowledge of Android app development, UI/UX design, Firebase integration, and project management. The application successfully delivers cultural information through an interactive mobile platform.
